# nav6 Components #

The nav6 IMU contains the following integrated circuits:

  * [Atmel ATMEGA328P Microcontroller](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=2&cad=rja&ved=0CDMQFjAB&url=http%3A%2F%2Fwww.atmel.com%2Fimages%2Fatmel-8271-8-bit-avr-microcontroller-atmega48a-48pa-88a-88pa-168a-168pa-328-328p_datasheet.pdf&ei=XEKzUp_MFJLkoATOvoCAAg&usg=AFQjCNG11iPfTyE8Bo48itJKyz_DwRYClg&sig2=Sfl8nCw1F9EZuvA3DVCyZQ&bvm=bv.58187178,d.cGU)
  * [Invensense MPU-6050 3-axis gyro, 3-axis accelerometer and motion processor](http://www.invensense.com/mems/gyro/documents/PS-MPU-6000A-00v3.4.pdf)
  * [Honeywell HMC5883L 3-axis magnetometer](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&ved=0CCwQFjAA&url=http%3A%2F%2Fwww51.honeywell.com%2Faero%2Fcommon%2Fdocuments%2Fmyaerospacecatalog-documents%2FDefense_Brochures-documents%2FHMC5883L_3-Axis_Digital_Compass_IC.pdf&ei=p0KzUsKdHYzooASKjYGgCg&usg=AFQjCNERVlsMXwth4SXIth4jIgNI8Y3gJg&sig2=fhM4QWNmQoZeSJniYJqfmw&bvm=bv.58187178,d.cGU)
  * [Texas Intruments MAX232 RS-232 Transceiver](http://www.ti.com/lit/ds/symlink/max232.pdf)
  * [Micrel 3975 3.3V and 5V DC Voltage Regulators](https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&ved=0CCwQFjAA&url=http%3A%2F%2Fwww.micrel.com%2F_PDF%2Fmic3975.pdf&ei=I0OzUsSZBdXeoASIqYCoBA&usg=AFQjCNG-YPsQa6S4Sk8jq_dBwrFd-Or2bg&sig2=XnxL6cOzmRuD483JNRHS0A&bvm=bv.58187178,d.cGU)
